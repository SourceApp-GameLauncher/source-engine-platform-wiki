# 🛠️ Source Engine 核心引擎更新


## v1.17.0026 (2024/01/26)
- 启动器修复：修复了所有游戏端口的启动器。
- 新构建发布：正式发布 1.17.0026 核心构建版本。
- Team Fortress 2 Classic：现在支持由 SanyaSho 制作的 TF2C 端口。
- 优化：为部分组件启用了 LTO (链接时优化)。
- 多人游戏：新增了专用的主服务器（仅限 .0026 构建）。


## v1.17.0025 (2024/01/24)
- 系统稳定性：修复了 IsMapValid 和 spec_goto 导致的崩溃。
- Android 专项修复：修复了最小化窗口后的黑屏和 VSync（垂直同步）问题。
- 性能参数：添加了 -nobackgroundlevel 启动参数。
- 渲染修复：修复了非 Windows 系统的颜色校正。
- 音频改进：音频现在在独立线程中运行。
- 底层变动：移除了 stlport，由 gnustl 替代；优化了 filesystem_async 线程计数。
- 触摸控制：改进了触摸响应灵敏度，修复了过场动画中触摸屏的透明度问题。


## v1.16 (2023/02/17)
- 多线程优化：为材质系统（Material System）添加了多线程支持。
- 64位支持：新增 WIN64 支持。
- 服务器功能：新增服务器浏览器（Server Browser）和主服务器支持。
- UI 调整：Android 端的控制台字体调小；主菜单新增 Discord、GitHub、Telegram 和 Twitter 按钮。
- 游戏设置：为 CS、TF、DOD 解锁了 -tickrate 参数。
- 地图修复：修复了加载 BSP20 地图时的 SOLID 标志错误。


## v1.14 (2022/09/19)
- 字体修复：解决了某些语言的字体显示问题。
- 触摸按钮：修复了触摸按钮的问题（ahsim 制作的 spawnmenu 现可正常工作）。
- 语言扩展：新增了泰语支持。


##v1.13 (2022/09/17)
- 内存优化：适配 64 位系统，解决了 4GB+ 内存设备上的“内存溢出”问题。
- 视觉渲染：新增 PBR（基于物理的渲染） 支持；支持 VTF 7.5 格式。
- 多语言：新增中/日/韩字体支持。
- 操控升级：最大 FOV 提高到 110；支持自定义触摸按钮的颜色和透明度。
- 物理修复：修复了玩家被物理道具卡住的问题，以及道具的保存/加载速度。
